# Merger Solution

> CODALAB SCORE: 0.43

## Description

This solution merges output from both the `dict` and `spacy-spancat` solutions.

## How to run

1. Make sure you have installed `weasel` by running `pip install weasel`.
2. Run the following command to execute the workflow:

```bash
weasel run merge
```
